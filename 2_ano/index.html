Sobre o Curso

O curso técnico de Desenvolvimento de Sistemas é focado em preparar os alunos para o mercado de trabalho, com conhecimentos práticos em linguagens de programação, levantamento de requisitos, lógica de programação, redes de computadores e muito mais. Este é o único curso técnico da rede pública de São Paulo voltado para o desenvolvimento de sistemas!

Disciplinas Oferecidas

  • Inteligência Artificial
  • Lógica de Programação com Python
  • Redes de Computadores e Segurança da Informação
  • Desenvolvimento de Software e Metodologias Ágeis
  • Versionamento de Código e Sistemas de Mensageria
  • Carreiras e Competências para o Mercado de Trabalho em Desenvolvimento de Sistemas

Participe do Nosso Curso!

Inscreva-se no curso técnico de Desenvolvimento de Sistemas da E.E. Prof. Amilcare Mattei e prepare-se para o futuro! Venha aprender as habilidades que o mercado de tecnologia exige e esteja pronto para iniciar sua carreira.
